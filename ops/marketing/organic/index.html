How This Works

This dashboard manages weekly organic posting for Nawabi Chai House across Instagram, Facebook, and Google Business.

Weekly workflow (one day of work):

  1. Review the 21 posts for the week (3 per day × 7 days)
  2. Generate all images in AI Studio using the prompts provided
  3. Upload each image and publish to all 3 platforms using the 4-step workflow
  4. Instagram & Facebook publish automatically from here
  5. Google Posts are manual (copy + paste into Google Business)

Upload 3 reference images to AI Studio (every time):

  • Image 1: NCH Logo — Brand logo (used in corner)
  • Image 2: Product photo — The food/drink photo for this post
  • Image 3: Menu card frame — Ornate heritage border reference (ensures every post has the same Mughal-style frame with corner flourishes on warm cream background)
