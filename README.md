# Nawabi Chai House Website

## Brand-Compliant Professional Website

### Quick Setup for Antigravity

1. **Import the folder** into Antigravity
2. **All assets are already named correctly** (underscores instead of spaces)
3. **Run preview**: `npx serve .`
4. **Open**: `http://localhost:3000`

### Asset Files Required

Place these files in the `/assets` folder:

| File Name | Purpose |
|-----------|---------|
| `NCH_Full_Logo_WO_BG.svg` | Full logo (hero, footer) |
| `NCH_Icon_WO_BG.svg` | Icon logo (favicon, loader, story section) |
| `NCH_Text_WO_BG.svg` | Text-only logo (navigation) |
| `NCH_Logo_Icon_With_Background.png` | Apple touch icon |

### GitHub Deployment

```bash
git init
git add .
git commit -m "Initial commit: Nawabi Chai House website"
git remote add origin https://github.com/YOUR_USERNAME/nawabi-chai-site.git
git push -u origin main
```

Then enable GitHub Pages or Cloudflare Pages.

### Customization

- Update address in Location section
- Replace Google Maps embed URL with actual location
- Update phone/email/WhatsApp links
- Add social media URLs
- Adjust menu items and prices
